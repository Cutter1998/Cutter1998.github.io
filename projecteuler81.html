Project Euler 81 in C++

To view or download the Matrix.txt file you can click the link below:

Matrix.txt

To solve this program using brute force would be incredibly inefficient. There are just too many combinations to try. We could however use dynamic programming. By storing the minimum cost of every single position in the matrix, we can recursively make our way back to position [0][0].

To work out the cost of a certain position, we must add to it to the smaller of the follwing values: the cost of the cell below it, or the cell to the right of it.

In the screenshot below I read in the Matrix.txt file and store the string values in a 1D vector. I will convert this to a 2D vector in the next screenshot.

Now I have the matrix in a string vector, I can convert it into a 2D integer vector.

Here is the slightly more complicated part. I iterate through every cell, starting at the bottom right, and calculate the minimum cost of the position should it be on the path selected. Obviously, the cost of the very last position will simply be the value of the element.

For every other position, we will use the formula: cost = valueOfElement + min(valueOfElementBelow, valueOfElementRight).

For edge cases, we have to choose the only route we can take.

This works recursively, all the way until we reach the very first element, whose new value should be the minimum cost of traversing the entire matrix via the optimal route.

Here we can see that the value of the first element will be displayed.
